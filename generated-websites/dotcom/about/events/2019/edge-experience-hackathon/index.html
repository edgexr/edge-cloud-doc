    Edge Experience Hackathon

    September 27, 2019 5:00 pm to September 28, 2019 10:00 pm

    Location: Kraków, Poland

    View the recap blog post here.

    Join MobiledgeX and partners for a 24-hour hackathon in Kraków, Poland! The challenge for the weekend is to create new immersive experiences using the MobiledgeX platform.

    Come hack on massive multiplayer games, machine learning (ML), and augmented reality (AR) apps that deliver the world you imagined as a kid. Whether you are interested in creating a larger-than-life dungeons and dragons AR game that runs on the latest Samsung S10 devices, a customer happiness detector that leverages open source ML libraries, real time ML video augmentation, or industrial automation with Vuzix Blade glasses. Sound like fun? RSVP Today!

    Applications need to use MobiledgeX to host the Docker / VM application. MobiledgeX employees
